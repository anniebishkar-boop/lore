# Плоскость Гранитум

Добро пожаловать в интерактивный мир «Плоскость Гранитум»!

Этот сайт — презентация вымышленного мира с изображениями, музыкой и озвучкой. Здесь собраны локации, материалы, источники энергии и персонажи нашей вселенной. Каждый объект сопровождается аудиофайлом, который можно прослушать, нажав на картинку.

В конце страницы вас ждёт сказка о мире, которую можно прочитать и прослушать диалог.

Сайт доступен по ссылке: https://anniebishkar-boop.github.io/lore/

Проект создан и поддерживается пользователем anniebishkar-boop. Все материалы являются частью творческого мира «Плоскость Гранитум» и защищены авторским правом.

Для запуска сайта просто откройте файл index.html в любом современном браузере. Для корректной работы требуется подключение к интернету, так как все медиафайлы загружаются с GitHub Pages.

Приятного погружения в мир Плоскости Гранитум!
